## Что сделано
Полный аналог «Аскан: Мониторинг транспорта» в `app/` (Next.js 16 + PostgreSQL 16, 165 роутов,
~40 тыс. строк): справочники + карточка клиента с операциями над парком, интеграция Omnicomm Online
(импорт абонбазы, консервация, двухэтапное удаление, автоблокировка должников), биллинг (посуточно
по истории состояний, иерархия тарифов, аванс/кредит, НДС-2026, двуязычные PDF: счёт/АВР Р-1/
расшифровка, ведомость, выгрузка 1С), сервисный контур (заявки → DnD-график → акты ТО → тестирование,
ремонт), PWA техника (/m: акт одним листом, подпись клиента, push), сдельная ЗП с порогом,
техподдержка (тикеты → авто-заявки, очередь уведомлений), отчёты + дашборд, i18n RU/KK.
